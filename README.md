# Artify

Base front initiale du projet APP Informatique.

## Contenu

- `index.html` : page d'accueil du projet
- `assets/css/styles.css` : styles de la page
- `assets/js/main.js` : interactions front

## Objectif

Cette premiere base reprend les consignes du document de specifications :

- mettre en avant la plateforme Artify
- presenter des creations et des artisans
- proposer une navigation simple
- offrir une recherche front simple sur la page d'accueil
- preparer une structure propre pour la suite du projet

## Lancer le projet

Ouvrir `index.html` dans un navigateur.
